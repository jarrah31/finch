Merchant Enrichment

Logo.dev API keys for merchant logos and brand name resolution

Used in page templates to fetch logos from the Logo.dev CDN: https://img.logo.dev/{domain}?token=pk_… Safe to include in frontend HTML.

Server-side only — never exposed to the browser. Used to call the Logo.dev Brand Search API to resolve merchant names to domains. Stored securely in the local database.

How to obtain Logo.dev API keys

  1. Go to logo.dev and create a free account
  2. Navigate to Dashboard → API Keys
  3. Copy both your Publishable key (pk_…) and Secret key (sk_…)

Free tier: 500,000 requests/month. The secret key is used server-side to look up merchant domains. The publishable key is used to display logos from the CDN.
